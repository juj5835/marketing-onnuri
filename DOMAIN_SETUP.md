# Onnuri Landing 도메인 연결 가이드 (domain.nhn.com 전용)

이 문서는 Onnuri Landing 페이지를 **onnuri-care.co.kr** 도메인과 연결하는 방법을 **domain.nhn.com** 사용자 환경에 맞춰 설명합니다.

---

## 🏗️ 방법 1: 네임서버 변경 (가장 쉽고 권장됨)

DNS 설정을 일일이 하기 어려울 때 사용하는 가장 확실한 방법입니다.

### [상세 순서]
1.  [domain.nhn.com](https://domain.nhn.com) -> 마이페이지 -> **[네임서버 변경]** 클릭
2.  **'타사 네임서버'** 또는 **'직접 입력'**을 선택합니다.
3.  아래의 Vercel 네임서버 주소를 차례대로 입력합니다: (⚠️ 주의: `.vercel.app` 주소가 아닙니다!)
    *   **1차 네임서버**: `ns1.vercel-dns.com` (IP: `198.51.44.13` 등 - 공란 가능 시 공란 추천)
    *   **2차 네임서버**: `ns2.vercel-dns.com` (IP: `198.51.45.13` 등 - 공란 가능 시 공란 추천)
4.  저장 버튼을 누릅니다. (반영까지 최대 24시간이 걸릴 수 있으나 보통 1시간 내에 완료됩니다.)

---

## ⚙️ 방법 2: DNS 레코드 설정 (수동 설정)

네임서버를 바꾸지 않고 값만 입력하고 싶을 때 사용합니다.

### [입력해야 할 레코드 값]

| 레코드 타입 | 호스트(Host) | 값(Value) |
| :--- | :--- | :--- |
| **A** | **@** | **76.76.21.21** |
| **CNAME** | **www** | **cname.vercel-dns.com.** |

### [상세 순서]
1.  마이페이지 -> `onnuri-care.co.kr` 우측의 **[관리]** 클릭
2.  **[부가서비스 관리] > [DNS 관리]** 클릭
3.  **[DNS 레코드 설정]** 버튼을 누르고 위의 표에 있는 값을 입력합니다.

---

## 3. 최종 확인 (Verification)

1.  Vercel의 Domains 페이지로 돌아와서 `Valid Configuration` (초록색 체크)이 뜨는지 확인합니다.
2.  반영이 완료되면 `https://onnuri-care.co.kr`로 접속이 가능해집니다.

> [!CAUTION]
> 1차 네임서버에 `marketing-onnuri.vercel.app`을 넣으면 절대 안 됩니다! 반드시 `ns1.vercel-dns.com`을 입력해 주세요.

---
